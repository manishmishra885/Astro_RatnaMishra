# Astrology & Numerology Website

A modern and responsive astrology website created for my mother, who is a well-known astrologer, numerologist, palm reader, and spiritual consultant with more than 25 years of experience.

## Live Website

🌐 https://sparkly-semifreddo-203f02.netlify.app/

## About The Project

This website was developed to create a strong online presence for astrology consultation services.  
The website focuses on clean UI, responsive design, smooth animations, and professional presentation.

It showcases:
- Astrology Services
- Numerology Consultation
- Palm Reading
- Kundli Guidance
- Spiritual Consultation
- Contact Information
- Social Media Integration

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Netlify (Deployment)

## Purpose

The goal of this project was to build a professional and trustworthy website for an experienced astrologer and help grow her online presence digitally.
